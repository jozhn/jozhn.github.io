SourceTree提交Github失败解决方法

发表于 2019-09-11

使用SourceTree提交代码到Github时会要求登录Github账号，但是输入正确的账号密码也会出现账号密码错误的提示，实际上并不是账号密码错了，而是Windows的SSL协议版本的问题。

出现该提示信息的主要原因是Github禁用了TLS v1.0 and v1.1这种弱加密标准，此时需要手动更新Windows的git凭证管理器，更新方式很简单，在网站下载Git在Window上的凭证管理器并默认安装即可，链接如下：
https://github.com/Microsoft/Git-Credential-Manager-for-Windows/releases/

下载其中的exe并安装就解决了。
